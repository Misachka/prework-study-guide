HTML

  • The head elements contains information about the webpage
  • The body element represents the visible content to show the user
  • Best to use semantic HTML in the body of the page to make the code easy to read and webpage accessible
  • metadata is information about the page that isn't displayed directly on the web page
  • Div container divides the page into sections

CSS

  • Cascadin style sheets (CSS) is a programming language that defines the colors, aesthetics,visual layout and organization of the website
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Do Not Repeat yourself (DRY) code is used to simplify code by combining selectors that follow the same rules

Git

  • git status: checks that branch we are currently on
  • git checkout -b branch-name: created a new branch and switched to it
  • pull system allows for multiple editors to double check ones work and merge code
  • code . -takes you to VS

JavaScript

  • A variable is a named container that allows us to store data in our code
  • control flow is the order in which a computer executes code in a script
  • Functions are a way to reuse code by defining a body of code that executes when you call the function name
  • IF statement is a way of introducing decision-making into our code
  • Array is a dingle variable that is used to hold a group data
